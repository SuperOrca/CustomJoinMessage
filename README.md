# CustomJoinMessage
 A lightweight plugin to customize joining and leaving messages!

## Versions Tested
* 1.16

## Configuration
```yml
# Welcome to the CustomJoinMessage config. Set to '' to disable.

join-message: '&e%player% joined the game.'
quit-message: '&e%player% left the game.'
join-title:
  enabled: true
  title: '&5&lWelcome to the Server!'
  subtitle: ''
  fadeIn: 20
  stay: 60
  fadeOut: 20
```
